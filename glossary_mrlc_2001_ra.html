    Entity ID
    Definition: This field uniquely identifies data which can be ordered in any combination of unique products.

     
    Format: IICSSPPPRRRMMDDYYYYXX
    II = DS ID
    C = Process Code
    SS = Landsat Satellite Number
    PPP = Path
    RRR = Row
    MMDDYYYY = Acquisition date (Month/Day/Year)
    XX = Version Number

    Satellite Number
    Definition: The number of the satellite which acquired the data.

     
    WRS-2 Path
    Definition: The WRS-2 Path number associated with the scene.

     
    Valid Values:
    4 through 81

    The Path is the longitudinal centerline of a Landsat scene, corresponding to the center of an orbital track. Sequential numbers from east to west are assigned for a total of 233 paths. The MRLC Dataset is a subset of 81 WRS-2 paths. The complete coverage cycle is 16 days. Path numbers can be used with Row numbers to designate a geographic search for a nominal scene center.

    The consistency and repeatability of the path for each satellite provides a common reference for all scenes acquired. The path is also a key indexing field for subsequent acquisition requests as well as data base inquiries.

    WRS-2 Row
    Definition: The WRS-2 Row number associated with the scene.

     
    Valid Values:
    26 through 47

    The Row is the latitudinal centerline of a Landsat scene. This nominal configuration for WRS-2 begins with row 1 at North 80 degrees, 47 minutes, and ends at row 122 at South 81 degrees, 51 minutes for daylight acquisitions. The MRLC Dataset is limited to rows 26-47.

    Acquisition Date
    Definition: The date the image was acquired

     
    Valid Values: DD/MMM/YY

    Date Entered
    Definition: The date the record for the image was created

     
    Valid Values: DD/MMM/YY

    Date Updated
    Definition: The date the record for the image was updated

     
    Valid Values: DD/MMM/YY

    Map Projection
    Definition: The name of the map projection used for the image

     
    Valid Values: Albers_Equal_Area

    Sun Elevation
    Definition: Elevation is the angle of vertical deviation, measured clockwise, of a bearing from a standard direction. The sun elevation angle at the "true" WRS-2 scene center

     
    Valid Values: -90.000000 through +90.000000 degrees

    Note: A positive value [+] indicates a daytime scene; a negative value [-] indicates a nighttime scene.

    (Note: TM (Landsat 4/5) values are precise only to the left of the decimal.)

    Sun Distance
    Definition: The distance between the Earth and the Sun in astronomical units (AU), a derived constant used to indicate distances within a solar system; the mean Earth orbit is at a distance of 1 AU from the Sun.

     
    Band Gain 1-6
    Definition: Detector gain state for the first data line (start) of a WRS scene.

     
    Band Bias 1-6
    Definition: Detector offset state for the first data line (start) of a WRS scene.

     
    Upper Left Corner
    Definition: Scene upper left corner latitude and longitude.

     
    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)
    SS = Seconds (Valid values: 0-59)
    H = Hemisphere (Valid values: N or S for Latitude, E or W for Longitude)

    Browse Available
    Definition: This flag identifies whether or not the scene has a browse image available. A browse image provides the user with a visual overview of an inventory item. This is an on-line digital (raster format) image of an individual scene.Note: Preview images are resampled from the original data and do not display the final MRLC product.

     
    Valid Values:
    Yes
    No
