    Entity ID
    Definition: This field uniquely identifies data which can be ordered in any combination of unique products.

     
    Format: IICSSPPPRRRMMDDYYYYXX
    II = DS ID
    C = Process Code
    SS = Landsat Satellite Number
    PPP = Path
    RRR = Row
    MMDDYYYY = Acquisition date (Month/Day/Year)
    XX = Version Number

    Satellite Number
    Definition : The number of the satellite which acquired the data.

     
    Dataset Type
    Definition: Type of data

     
    Format: (data source)_(sensor or data type)
    Valid Values:
    EDC_TM
    EDC_ETM+

    WRS-2 Path
    Definition: The WRS-2 Path number associated with the scene.

     
    Valid Values:
    4 through 89

    The Path is the longitudinal centerline of a Landsat scene, corresponding to the center of an orbital track. Sequential numbers from east to west are assigned for a total of 233 paths. The MRLC Dataset is a subset of 89 WRS-2 paths. The complete coverage cycle is 16 days. Path numbers can be used with Row numbers to designate a geographic search for a nominal scene center.

    The consistency and repeatability of the path for each satellite provides a common reference for all scenes acquired. The path is also a key indexing field for subsequent acquisition requests as well as data base inquiries.

    WRS-2 Row
    Definition: The WRS-2 Row number associated with the scene.

     
    Valid Values:
    10 through 47

    The Row is the latitudinal centerline of a Landsat scene. This nominal configuration for WRS-2 begins with row 1 at North 80 degrees, 47 minutes, and ends at row 122 at South 81 degrees, 51 minutes for daylight acquisitions. The MRLC Dataset is limited to rows 10-47.

    Acquisition Date
    Definition: The date the image was acquired

     
    Valid Values: DD/MMM/YY

    Date Entered
    Definition: The date the record for the image was created

     
    Valid Values: DD/MMM/YY

    Date Updated
    Definition: The date the record for the image was updated

     
    Valid Values: DD/MMM/YY

    Map Projection
    Definition: The name of the map projection used for the image

     
    Valid Values: Albers_Equal_Area

    Horiz Datum
    Definition: An abbreviation of the datum used

     
    Valid Values:
    NAD83
    WGS84

    Resampling
    Definition: The type of resampling used with the image.

     
    Valid Values:
    CC = cubic convolution
    NN = nearest neighbor

    Pixel Units
    Definition: The units of measure for pixel size

     
    Valid Values:
    M = Meters

    USGS Projection Number
    Definition: USGS supported projection number, as specified in the General Cartographic Transformation Package document.

     
    Valid Values:
    1 = UTM
    3 = AEA

    Ellipsoid Semi Major
    Definition: Semi-major axis of earth ellipsoid in meters

     
    Ellipsoid Semi Minor
    Definition: Semi-minor axis of earth ellipsoid in meters

     
    Sun Elevation
    Definition: Elevation is the angle of vertical deviation, measured clockwise, of a bearing from a standard direction. The Sun elevation angle at the "true" WRS-2 scene center.

     
    Valid values:
    -90.000000 through +90.000000 degrees

    Note: A positive value [+] indicates a daytime scene; a negative value [-] indicates a nighttime scene.

    (Note: TM (Landsat 4/5) values are precise only to the left of the decimal.)
     
    Sun Azimuth
    Definition: Azimuth is the angle of horizontal deviation, measured clockwise, of a bearing from a standard direction. The Sun azimuth angle at the "true" WRS-2 scene center.

    Valid values:
    -180.000000 through +180.000000 degrees

    Note: A positive value [+] indicates angles to the East or clockwise from North; a negative value [-] indicates angles to the West or counter-clockwise from North.

    (Note: TM (Landsat 4/5) values are precise only to the left of the decimal.)
     
    Upper Left Corner
    Definition: Scene upper left corner latitude and longitude.

     
    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)
    SS = Seconds (Valid values: 0-59)
    H = Hemisphere (Valid values: N or S for Latitude, E or W for Longitude)

    Upper Right Corner

    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)

    SS = Seconds (Valid values: 0-59)

    USGS Projection Parameters 1-15
    Definition: USGS map projection parameters. The map projection are floating point numbers that represent different parameters for different projections (not detailed here).

     
    Browse Available
    Definition: This flag identifies whether or not the scene has a browse image available. A browse image provides the user with a visual overview of an inventory item. This is an on-line digital (raster format) image of an individual scene. (Note: Preview images are resampled from the original data and do not display the final MRLC product.)

     
    Valid Values:
    Yes
    No
