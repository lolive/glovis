    L1A Scene ID, L1B Scene ID
    Definition: This field is the DAAC Level 1A or Level 1B scene ID. It uniquely identifies the scene for ordering it from the DAAC.

     
    Format: SC:AST_L1A.VVV:XXXXXXXXXX
    VVV Database version
    XXXXXXXXXX Unique database ID for the specified version

     
    Acquisition Date
    Definition: The date the image was acquired.

     
    Format: YYYY/MM/DD

     
    WRS-2 Path
    Definition: The WRS-2 Path number associated with the scene.

     
    Valid Values:
    001 through 233
    The Path is the WRS-2 path into which the ASTER scene falls. Sequential numbers from east to west are assigned to 233 nominal satellite tracks. Path numbers can be used with Row numbers to designate a geographic search for a nominal scene center.

    The consistency and repeatability of the path for each satellite provides a common reference for all scenes acquired.

    Note: The ASTER scene centers do not exactly follow a WRS-2 layout. The Global Visualization Viewer uses the WRS-2 Path/Row for data organization.



     
    WRS-2 Row
    Definition: The WRS-2 Row number associated with the scene.

     
    Valid Values:
    001 through 248
    The Row is the WRS-2 row into which the ASTER scene falls. This nominal configuration begins with row 1 at North 80 degrees 47 minutes and ends at row 122 at South 81 degrees 51 minutes for daylight acquisitions. Row 60 corresponds with the equator at 0 degrees latitude. There is a total of 248 rows. Rows 123-248 are associated with night acquisitions.

    Note: The ASTER scene centers do not exactly follow a WRS-2 layout. The Global Visualization Viewer uses the WRS-2 Path/Row for data organization.



     
    Upper Left Corner
    Definition: Scene upper left corner latitude and longitude.

     
    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)
    SS = Seconds (Valid values: 0-59)
    H = Hemisphere (Valid values: N or S for Latitude, E or W for Longitude)

     
    Upper Right Corner
    Definition: Scene upper right corner latitude and longitude.

     
    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)
    SS = Seconds (Valid values: 0-59)
    H = Hemisphere (Valid values: N or S for Latitude, E or W for Longitude)

     
    Lower Left Corner

    SS = Seconds (Valid values: 0-59)

    Lower Right Corner

    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)

    Scene Center
    Definition: WRS-2 Scene Center Latitude/Longitude - DAAC calculated coordinate value.

     
    Format: DDD°MM'SS"H
    DDD = Degrees (Valid values: 0-90 for Latitude, 0-180 for Longitude)
    MM = Minutes (Valid values: 0-59)
    SS = Seconds (Valid values: 0-59)
    H = Hemisphere (Valid values: N or S for Latitude, E or W for Longitude)

     
    Scene Cloud Cover
    Definition:The Scene Cloud Cover is the percentage of a scene that is obscured by clouds and is the average of all four quadrant cloud cover values. Users are encouraged to preview browses prior to ordering data products to confirm acceptable cloud cover impacts.


     
    SWIR Mode
    Definition: This flag indicates whether or not the shortwave-infrared sensor collected data for this scene.

     
    Valid Codes:
    On
    Off

    Note: Browse imagery is not available in the Global Visualization Viewer.

     
    TIR Mode
    Definition: This flag indicates whether or not the thermal-infrared sensor collected data for this scene.

     
    Valid Codes:
    On
    Off

    Note: Browse imagery is not available in the Global Visualization Viewer.

     
    VNIR1 Mode
    Definition: This flag indicates whether or not the visible, near-infrared sensor for telescope #1 collected data for this scene.

     
    Valid Codes:
    On
    Off

    Note: Browse imagery is  available in the Global Visualization Viewer. The single set of browse images in the Global Visualization Viewer is an integration of the VNIR1 and VNIR2 scenes.

     
    VNIR2 Mode
    Definition: This flag indicates whether or not the visible, near-infrared sensor for telescope #2 collected data for this scene.

     
    Valid Codes:
    On
    Off

    Note: Browse imagery is available in the Global Visualization Viewer. The single set of browse images in the Global Visualization Viewer is an integration of the VNIR1 and VNIR2 scenes.

     
    Day or Night
    Definition: This field indicates whether the scene was collected during the day or night. If a collected scene crosses the sun terminator line, it is declared as "Both".

     
    Valid Codes:
    Day
    Night
    Both

     
    Orbital Direction
    Definition: This field indicates the orbital direction for the scene when the data was acquired. Terra is a sun-synchronous orbit, crossing the equator at about 10:30 AM (GMT). This renders day-time orbits to be descending passes while night-time orbits are ascending passes.

     
    Valid Codes:
    Ascending
    Descending

     
    Sun Elevation
    Definition: Elevation is the angle of vertical deviation, measured clockwise, of a bearing from a standard direction.

     
    Valid Values:
    -90.000000 through +90.000000 degrees

    Note: A positive value [+] indicates a daytime scene; a negative value [-] indicates a nighttime scene. 

     
    Sun Azimuth
    Definition: Azimuth is the angle of horizontal deviation, measured clockwise, of a bearing from a standard direction.

     
    Valid Values:
    -180.000000 through +180.000000 degrees

    Note: A positive value [+] indicates angles to the East or clockwise from North; a negative value [-] indicates angles to the West or counter-clockwise from North. 

     
    Acquisition Time
    Definition: The hour, minute, and second at which the satellite collected the data for the scene.

     
    Format: HH:MM:SS
    HH = hours (00-23)
    MM = minutes (00-59)
    SS = second (00-59)

    VNIR, TIR or SWIR Pointing Angle
    Definition: The VNIR, TIR or SWIR sensor pointing angle in degrees from nadir.

    ASTER Granule ID
    Definition: Granule ID assigned by the Japanese acquisition system.

    ASTER Processing Center
    Definition: Center that processed the granule.
